# تصميم الواجهات الأمامية (Frontend Design)

**المصدر:** `anthropics/skills/frontend-design`
**التصنيف:** AI & LLM Integration
**التثبيت:** npx skills add anthropics/skills/frontend-design

## الوصف
مهارة لتصميم وتطوير واجهات أمامية جميلة وعملية باستخدام أفضل ممارسات التصميم الحديث والذكاء الاصطناعي.

## الاستخدام
تستخدم لتصميم واجهة مستخدم جذابة واحترافية لمشروع KING2 AI مع تحسين تجربة المستخدم.
