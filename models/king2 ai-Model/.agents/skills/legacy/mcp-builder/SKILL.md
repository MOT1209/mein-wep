# باني MCP (MCP Builder)

**المصدر:** `anthropics/skills/mcp-builder`
**التصنيف:** AI & LLM Integration
**التثبيت:** npx skills add anthropics/skills/mcp-builder

## الوصف
أداة لبناء وإنشاء خوادم MCP (Model Context Protocol) لتوسيع قدرات الـ AI agent وربطه بخدمات خارجية.

## الاستخدام
تستخدم لإنشاء خوادم MCP مخصصة لربط KING2 AI بقواعد بيانات و APIs وخدمات خارجية.
