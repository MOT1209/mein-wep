# تسليم المهام (Handoff)

**المصدر:** `mattpocock/skills/handoff`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add mattpocock/skills/handoff

## الوصف
مهارة تنظيم وتسليم المهام بين المطورين أو بين جلسات العمل، مع توثيق الحالة الحالية والخطوات التالية.

## الاستخدام
تستخدم لتوثيق حالة مشروع KING2 AI عند تسليم المهام بين أعضاء الفريق أو بين الجلسات.
