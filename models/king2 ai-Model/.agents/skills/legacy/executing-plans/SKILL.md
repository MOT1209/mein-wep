# تنفيذ الخطط (Executing Plans)

**المصدر:** `obra/superpowers/executing-plans`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add obra/superpowers/executing-plans

## الوصف
مهارة لتنفيذ خطط التطوير بكفاءة، مع متابعة التقدم وتعديل المسار حسب الحاجة لضمان إنجاز المهام.

## الاستخدام
تستخدم لتنفيذ خطط تطوير KING2 AI بشكل منهجي وضمان إكمال المهام وفق الجدول.
