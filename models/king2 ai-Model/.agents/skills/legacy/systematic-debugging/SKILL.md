# التصحيح المنهجي (Systematic Debugging)

**المصدر:** `obra/superpowers/systematic-debugging`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add obra/superpowers/systematic-debugging

## الوصف
منهجية منظمة ومنهجية لتصحيح الأخطاء البرمجية، مع خطوات واضحة لتحديد وعزل وحل المشاكل.

## الاستخدام
تطبق هذه المنهجية في KING2 AI لحل الأخطاء البرمجية المعقدة بطريقة منهجية وفعالة.
