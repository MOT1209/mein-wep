# النشر على Vercel (Deploy to Vercel)

**المصدر:** `vercel-labs/agent-skills/deploy-to-vercel`
**التصنيف:** Core Development
**التثبيت:** npx skills add vercel-labs/agent-skills/deploy-to-vercel

## الوصف
مهارة نشر التطبيقات على منصة Vercel بسهولة. تغطي الإعدادات المثلى والنشر المستمر والتكوين للمشاريع الحديثة.

## الاستخدام
استخدم هذه المهارة لنشر مشروع KING2 AI على Vercel مع أفضل الإعدادات والأداء العالي.
