# أساسيات Firebase (Firebase Basics)

**المصدر:** `firebase/agent-skills/firebase-basics`
**التصنيف:** Developer Tools
**التثبيت:** npx skills add firebase/agent-skills/firebase-basics

## الوصف
مهارة أساسيات استخدام Firebase، تشمل Firestore والتخزين والوظائف السحابية وإعداد المشروع.

## الاستخدام
تستخدم لإعداد وربط مشروع KING2 AI مع Firebase للخدمات الخلفية.
