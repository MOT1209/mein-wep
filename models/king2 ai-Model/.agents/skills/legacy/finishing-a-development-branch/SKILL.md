# إنهاء فرع تطويري (Finishing a Development Branch)

**المصدر:** `obra/superpowers/finishing-a-development-branch`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add obra/superpowers/finishing-a-development-branch

## الوصف
مهارة لإكمال ودمج فروع التطوير بشكل صحيح، مع مراجعة الكود وحل التعارضات وضمان الجودة.

## الاستخدام
تستخدم لإنهاء فروع التطوير في مشروع KING2 AI ودمجها مع الفرع الرئيسي بشكل آمن.
