# التطوير بالوكيل الفرعي (Subagent-Driven Development)

**المصدر:** `obra/superpowers/subagent-driven-development`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add obra/superpowers/subagent-driven-development

## الوصف
منهجية تطوير تستخدم وكلاء فرعيين (subagents) للعمل بالتوازي على مهام مختلفة لزيادة الإنتاجية.

## الاستخدام
تستخدم في KING2 AI لتوزيع المهام على وكلاء فرعيين للعمل المتوازي وتسريع التطوير.
