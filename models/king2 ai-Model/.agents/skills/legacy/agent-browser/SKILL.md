# متصفح الوكيل (Agent Browser)

**المصدر:** `vercel-labs/agent-browser/agent-browser`
**التصنيف:** AI & LLM Integration
**التثبيت:** npx skills add vercel-labs/agent-browser/agent-browser

## الوصف
متصفح آلي يمكن الـ AI agent من التفاعل مع صفحات الويب، تنفيذ مهام، وجمع بيانات من المتصفح بشكل كامل.

## الاستخدام
يستخدم لتزويد KING2 AI بقدرة تصفح الويب وتنفيذ مهام تفاعلية على المواقع الخارجية.
