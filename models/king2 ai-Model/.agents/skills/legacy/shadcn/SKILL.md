# Shadcn UI (Shadcn)

**المصدر:** `shadcn/ui/shadcn`
**التصنيف:** UI/UX Design
**التثبيت:** npx skills add shadcn/ui/shadcn

## الوصف
مكتبة مكونات واجهة مستخدم جميلة وقابلة للتخصيص، مبنية على Radix UI و Tailwind CSS مع تجربة مطور ممتازة.

## الاستخدام
تستخدم لإضافة مكونات واجهة جاهزة وقابلة للتخصيص لمشروع KING2 AI.
