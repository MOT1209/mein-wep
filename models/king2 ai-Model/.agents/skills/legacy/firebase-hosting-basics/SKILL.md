# أساسيات استضافة Firebase (Firebase Hosting Basics)

**المصدر:** `firebase/agent-skills/firebase-hosting-basics`
**التصنيف:** Developer Tools
**التثبيت:** npx skills add firebase/agent-skills/firebase-hosting-basics

## الوصف
مهارة نشر واستضافة تطبيقات الويب على Firebase Hosting، مع تكوين النطاق والشهادة والأداء.

## الاستخدام
تستخدم لنشر واستضافة مشروع KING2 AI على Firebase Hosting بسرعة وأمان.
