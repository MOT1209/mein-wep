# إرشادات تصميم الويب (Web Design Guidelines)

**المصدر:** `vercel-labs/agent-skills/web-design-guidelines`
**التصنيف:** Core Development
**التثبيت:** npx skills add vercel-labs/agent-skills/web-design-guidelines

## الوصف
مبادئ وإرشادات تصميم مواقع الويب الحديثة من Vercel، تشمل التخطيط والألوان والطباعة وتجربة المستخدم لتطبيقات Next.js.

## الاستخدام
تطبق إرشادات التصميم لتحسين واجهة مستخدم مشروع KING2 AI وجعلها احترافية وجذابة.
