# كتابة مهارة (Write a Skill)

**المصدر:** `mattpocock/skills/write-a-skill`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add mattpocock/skills/write-a-skill

## الوصف
مهارة تعليمية لكتابة وتطوير مهارات جديدة للنظام، مع شرح الهيكل والمكونات وأفضل الممارسات.

## الاستخدام
تستخدم كمرجع لإنشاء مهارات مخصصة لـ KING2 AI باتباع أفضل الممارسات.
