# استخلاص (Distill)

**المصدر:** `pbakaus/impeccable/distill`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add pbakaus/impeccable/distill

## الوصف
مهارة لاستخلاص وتلخيص المعلومات المهمة من قاعدة الكود، مع تقديم رؤى واضحة وقابلة للتنفيذ.

## الاستخدام
تستخدم لاستخلاص رؤى ومعلومات قيمة من مشروع KING2 AI لاتخاذ قرارات أفضل.
