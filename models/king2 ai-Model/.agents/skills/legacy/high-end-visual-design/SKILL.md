# التصميم البصري المتقدم (High-End Visual Design)

**المصدر:** `leonxlnx/taste-skill/high-end-visual-design`
**التصنيف:** UI/UX Design
**التثبيت:** npx skills add leonxlnx/taste-skill/high-end-visual-design

## الوصف
مهارة للتصميم البصري الراقي مع تقنيات متقدمة في الإضاءة والظلال والتدرجات والتأثيرات البصرية.

## الاستخدام
تستخدم لإضافة لمسات تصميمية راقية ومتقدمة لواجهة مشروع KING2 AI.
