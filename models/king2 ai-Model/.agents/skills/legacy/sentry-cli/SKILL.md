# Sentry CLI (Sentry CLI)

**المصدر:** `sentry/dev/sentry-cli`
**التصنيف:** Core Development
**التثبيت:** npx skills add sentry/dev/sentry-cli

## الوصف
أداة سطر الأوامر لإدارة ومراقبة الأخطاء في تطبيقات الإنتاج باستخدام Sentry، تشمل تتبع الأعطال والأداء.

## الاستخدام
استخدم هذه الأداة لربط ومراقبة تطبيق KING2 AI مع Sentry لتتبع الأخطاء وتحسين الاستقرار.
