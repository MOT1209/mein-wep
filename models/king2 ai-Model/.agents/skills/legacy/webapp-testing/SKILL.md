# اختبار تطبيقات الويب (Webapp Testing)

**المصدر:** `anthropics/skills/webapp-testing`
**التصنيف:** AI & LLM Integration
**التثبيت:** npx skills add anthropics/skills/webapp-testing

## الوصف
مهارة شاملة لاختبار تطبيقات الويب باستخدام الذكاء الاصطناعي، تشمل اختبارات الوحدة والتكامل والنهاية إلى النهاية.

## الاستخدام
تستخدم لكتابة وتنفيذ اختبارات تلقائية لمشروع KING2 AI لضمان الجودة والاستقرار.
