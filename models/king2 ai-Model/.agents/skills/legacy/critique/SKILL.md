# نقد (Critique)

**المصدر:** `pbakaus/impeccable/critique`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add pbakaus/impeccable/critique

## الوصف
أداة للنقد البناء وتحليل الكود، مع تقديم ملاحظات موضوعية لتحسين الجودة والأداء والتصميم.

## الاستخدام
تستخدم للحصول على نقد بناء وتحليل موضوعي لكود مشروع KING2 AI.
