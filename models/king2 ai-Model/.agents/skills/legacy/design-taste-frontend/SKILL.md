# ذوق التصميم للواجهات الأمامية (Design Taste Frontend)

**المصدر:** `leonxlnx/taste-skill/design-taste-frontend`
**التصنيف:** UI/UX Design
**التثبيت:** npx skills add leonxlnx/taste-skill/design-taste-frontend

## الوصف
مهارة لتحسين الذوق التصميمي للواجهات الأمامية، مع إرشادات حول الألوان والخطوط والمسافات والتخطيط.

## الاستخدام
تستخدم لتحسين جماليات واجهة KING2 AI ورفع مستوى الذوق التصميمي.
