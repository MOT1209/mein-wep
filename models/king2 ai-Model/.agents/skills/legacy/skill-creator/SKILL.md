# منشئ المهارات (Skill Creator)

**المصدر:** `anthropics/skills/skill-creator`
**التصنيف:** AI & LLM Integration
**التثبيت:** npx skills add anthropics/skills/skill-creator

## الوصف
أداة لإنشاء وتطوير مهارات جديدة للـ AI agent بسهولة، مع قوالب وأفضل الممارسات من Anthropic.

## الاستخدام
تستخدم لإنشاء مهارات مخصصة لـ KING2 AI لتوسيع قدراته وتخصيص سلوكه حسب احتياجات المشروع.
