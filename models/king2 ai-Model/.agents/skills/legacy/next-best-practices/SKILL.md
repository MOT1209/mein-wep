# أفضل ممارسات Next.js (Next.js Best Practices)

**المصدر:** `vercel-labs/next-skills/next-best-practices`
**التصنيف:** Core Development
**التثبيت:** npx skills add vercel-labs/next-skills/next-best-practices

## الوصف
أفضل الممارسات الرسمية لتطوير تطبيقات Next.js، تشمل Server Components والتحميل والتحسين والأداء والتوجيه.

## الاستخدام
اتبع هذه الإرشادات في مشروع KING2 AI عند العمل مع Next.js لضمان أفضل أداء وتجربة مستخدم.
