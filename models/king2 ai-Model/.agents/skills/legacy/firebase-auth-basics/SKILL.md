# أساسيات مصادقة Firebase (Firebase Auth Basics)

**المصدر:** `firebase/agent-skills/firebase-auth-basics`
**التصنيف:** Developer Tools
**التثبيت:** npx skills add firebase/agent-skills/firebase-auth-basics

## الوصف
مهارة شاملة لنظام المصادقة في Firebase، مع طرق تسجيل الدخول المختلفة وإدارة المستخدمين.

## الاستخدام
تستخدم لإعداد نظام مصادقة آمن لمشروع KING2 AI باستخدام Firebase Auth.
