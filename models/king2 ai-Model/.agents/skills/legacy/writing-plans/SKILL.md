# كتابة الخطط (Writing Plans)

**المصدر:** `obra/superpowers/writing-plans`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add obra/superpowers/writing-plans

## الوصف
مهارة لكتابة وتوثيق خطط تطوير واضحة وقابلة للتنفيذ، مع تحديد الأهداف والخطوات والجداول الزمنية.

## الاستخدام
تستخدم لإنشاء خطط تطوير منظمة لمشروع KING2 AI قبل بدء مراحل التطوير.
