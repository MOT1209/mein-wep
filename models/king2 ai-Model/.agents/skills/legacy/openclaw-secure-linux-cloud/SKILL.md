# السحابة الآمنة مع Linux (OpenCLAW Secure Linux Cloud)

**المصدر:** `xixu-me/skills/openclaw-secure-linux-cloud`
**التصنيف:** Developer Tools
**التثبيت:** npx skills add xixu-me/skills/openclaw-secure-linux-cloud

## الوصف
مهارة لإعداد وتأمين خوادم Linux على السحابة، مع أفضل ممارسات الأمان والتكوين والمراقبة.

## الاستخدام
تستخدم لإعداد البنية التحتية السحابية الآمنة لتطبيق KING2 AI.
