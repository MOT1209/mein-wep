# أفضل ممارسات Better Auth (Better Auth Best Practices)

**المصدر:** `better-auth/skills/better-auth-best-practices`
**التصنيف:** Core Development
**التثبيت:** npx skills add better-auth/skills/better-auth-best-practices

## الوصف
إرشادات وأفضل الممارسات لاستخدام Better Auth في المصادقة والتفويض، مع تغطية الأمان والجلسات وإدارة المستخدمين.

## الاستخدام
تطبق هذه الممارسات لنظام المصادقة في مشروع KING2 AI لضمان أمان عالي وتجربة مستخدم سلسة.
