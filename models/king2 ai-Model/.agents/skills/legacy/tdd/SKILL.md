# التطوير الموجه بالاختبارات (TDD)

**المصدر:** `mattpocock/skills/tdd`
**التصنيف:** Software Architecture
**التثبيت:** npx skills add mattpocock/skills/tdd

## الوصف
منهجية التطوير الموجه بالاختبارات (Test-Driven Development) لكتابة كود موثوق وقابل للاختبار مع ضمان الجودة العالية.

## الاستخدام
تطبق منهجية TDD في تطوير مشروع KING2 AI لضمان كتابة كود نظيف وخالٍ من الأخطاء.
