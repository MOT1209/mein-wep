# 🎨 كيف تشغّل نموذج KING2-IMAGE المدرّب مجاناً

نموذجك المدرّب محفوظ على HuggingFace: **`RASHID778/king2-image`** (محوّل الستيكمان في مسار `stickman/`).
يمكنك تشغيله مجاناً على GPU مجاني (Google Colab أو Kaggle) في أي وقت.

> **مهمّ:** استخدم دائماً **قوة LoRA = 0.6** ولاحقة التفعيل `, simple black and white stick figure line drawing`.
> هذا ما أثبتته اختباراتنا لأفضل جودة (القوة 1.0 تعطي فوضى خطوط).

## الطريقة 1 — Google Colab (الأسهل، GPU مجاني) ⭐

1. افتح https://colab.research.google.com → **New notebook**
2. من القائمة: **Runtime → Change runtime type → T4 GPU → Save**
3. الصق هذا الكود في خلية واضغط تشغيل (▶). أول مرة تأخذ ~3-4 دقائق (تنزيل النموذج)، ثم كل صورة ~10 ثوانٍ:

```python
!pip install -q diffusers transformers accelerate peft safetensors

import torch
from diffusers import DiffusionPipeline

pipe = DiffusionPipeline.from_pretrained(
    "stabilityai/stable-diffusion-xl-base-1.0",
    torch_dtype=torch.float16, use_safetensors=True,
).to("cuda")
pipe.load_lora_weights(
    "RASHID778/king2-image",
    weight_name="stickman/pytorch_lora_weights.safetensors",
    adapter_name="stickman",
)
pipe.set_adapters(["stickman"], adapter_weights=[0.6])   # ← القوة المهمة

def draw(prompt):
    img = pipe(
        prompt + ", simple black and white stick figure line drawing",
        num_inference_steps=25, guidance_scale=8.0, height=768, width=768,
    ).images[0]
    return img

draw("a stickman running")          # جرّب أي وصف
```

4. غيّر النص داخل `draw("...")` لأي وضع تريده:
   `a stickman jumping` · `a stickman kicking a ball` · `a stickman doing yoga` …

## الطريقة 2 — Kaggle (GPU مجاني، 30 ساعة أسبوعياً)

السكربت جاهز في: `kaggle_infer/king2_stickman_infer.py`

1. ادخل https://www.kaggle.com/code → **New Notebook**
2. من اليمين: **Settings → Accelerator → GPU T4 ×2**، و **Internet → On**
3. الصق محتوى `king2_stickman_infer.py` واضغط **Run All**
4. يولّد شبكة 9 صور بأوضاع مختلفة، تجدها في مخرجات النوتبوك (`stickman_grid.png`)

> نفس فكرة Colab لكن على Kaggle. ملف `kernel-metadata.json` بجانبه يتيح رفعه عبر أمر
> `kaggle kernels push` إن أردت الأتمتة.
