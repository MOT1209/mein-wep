How to Install

Since these apps are distributed directly (not via Google Play), Android shows a one-time security prompt. Just follow these simple steps:

  1. 1

    Download the APK

    Tap Download APK on any app above. The file saves to your device.

  2. 2

    Verify the file (optional but recommended)

    Each APK page shows its SHA-256 fingerprint. Click it to copy the full hash. Use a hash checker app like Hash Droid to confirm the file matches — this guarantees the file hasn't been tampered with.

  3. 3

    Open the file

    Tap the download notification, or find the file in your Downloads / Files app and open it.

  4. 4

    Allow "Unknown Sources"

    Android may warn that installing is blocked. Tap Settings, then enable Allow from this source (also called "Install unknown apps") for your browser or file manager.

  5. 5

    Tap Install

    Go back and press Install. Once it finishes, open the app and you're ready to go!

The "Unknown Sources" warning is standard for any app installed outside Google Play — it does not mean the app is unsafe. You can turn the permission back off after installing.
