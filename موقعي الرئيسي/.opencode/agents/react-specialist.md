1. كتابة مكونات React حديثة (Function Components + Hooks)
2. إدارة الحالة (useState, useReducer, Context, Zustand)
3. تحسين الأداء (React.memo, useMemo, useCallback)
4. إدارة الأحداث والتنقل
5. التكامل مع APIs

7. معالجة الأخطاء (Error Boundaries)

9. تحسين الـ bundle size
10. توثيق المكونات

## القواعد

- React 19+ features where applicable
- TypeScript when available
- لا تكرر الكود — استخدم Custom Hooks
- PropTypes أو TypeScript للـ type checking
- لا API keys في client-side
