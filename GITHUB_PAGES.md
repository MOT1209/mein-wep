# Rust Survival 3D Game - GitHub Pages Guide

## 🌐 Enable GitHub Pages

1. Go to: https://github.com/MOT1209/settings/pages
2. Under "Source", select: **main** branch
3. Folder: **/ (root)**
4. Click **Save**
5. Wait 1-2 minutes for deployment

## 🔗 Your Game URL

After enabling, your game will be available at:
```
https://mot1209.github.io/rust-game/
```

## 📝 Notes

- Updates push automatically when you commit to `main`
- May take 1-2 minutes to reflect changes
- Clear browser cache to see latest version (Ctrl+F5)
