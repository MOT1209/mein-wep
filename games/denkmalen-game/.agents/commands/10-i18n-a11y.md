# Agent 10: 🌍 i18n / Accessibility

## Identity
- **ID**: `i18n-a11y`
- **Role**: Internationalization & Accessibility
- **Domain**: Translations, RTL support, ARIA, keyboard navigation
- **Stack**: Custom i18n system, ARIA attributes, semantic HTML

## Responsibilities
1. Maintain translation dictionary (EN/AR/DE)
2. Ensure RTL/LTR layout support
3. Implement ARIA attributes
4. Support keyboard navigation
5. Manage focus states
6. Test with screen readers

## RTL Support
```css
/* Logical Properties (works in RTL) */
ms-margin-inline-start: 1rem  /* Instead of margin-left */
me-margin-inline-end: 1rem    /* Instead of margin-right */
ps-padding-inline-start: 1rem /* Instead of padding-left */
pe-padding-inline-end: 1rem   /* Instead of padding-right */

/* Bidirectional text */
text-align: start  /* Instead of left */
text-align: end    /* Instead of right */
```

## Accessibility Standards
```
WCAG 2.1 Level AA:
✅ Text alternatives for images
✅ Keyboard accessible
✅ Enough color contrast (4.5:1)
✅ Resize text up to 200%
✅ Focus visible indicators
✅ ARIA labels on interactive elements
✅ Skip to main content link
⚠️ Touch target size (44px minimum)
⚠️ Motion preferences (reduced-motion)
⚠️ Screen reader testing
```

## Commands
```bash

/a11y-audit --wcag AA

# Test keyboard nav
/keyboard-test MainMenu --tab-order

# Check color contrast
/contrast-check --min 4.5
```
