Featured Project

KingCraft

A 3D voxel sandbox game built with Three.js — mine, craft, build, and survive!

Play Now Source Code

🚀 Features

Voxel World

Infinite procedurally generated world made of blocks. Each block is a voxel in a 16×16×64 chunk system.

Physics Engine

Full collision detection, gravity, jumping, sprinting, and flying mode. Smooth player movement with pointer lock.

Block Interaction

Place and break blocks with raycasting. Left click to break, right click to place. Hotbar with 9 slots.

Performance

GPU-instanced rendering, chunk culling, and greedy meshing. Runs at 60fps on modern hardware.

Keyboard Controls

WASD movement, Space to jump, Shift to sprint, E for inventory, F3 for debug overlay, and more.

Open Source

Fully open source under the main mein-wep repository. Built with vanilla JavaScript and Three.js r160.
