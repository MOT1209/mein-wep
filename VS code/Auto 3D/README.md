# 3D Car Game - لعبة السيارة 3D

A fun 3D car racing game built with Three.js and Cannon.js physics engine.

## Features

- 3D car with realistic physics
- Interactive controls
- Racing environment with buildings
- Dynamic camera following
- Speed display
- Arabic/English UI support

## Controls

- **Arrow Keys / WASD**: Move car
- **Space**: Brake
- **Mouse**: Look around

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Technologies Used

- **Three.js**: 3D graphics
- **Cannon.js**: Physics engine
- **TypeScript**: Type-safe JavaScript
- **Vite**: Fast build tool

## Game Controls

- Forward/Backward: Use arrow keys or WASD
- Turn Left/Right: Use arrow keys or AD
- Brake: Press Space

حكم اللعبة:
- الامام / الخلف: استخدم أسهم الاتجاهات أو WASD
- الانعطاف اليسار / اليمين: استخدم أسهم الاتجاهات أو AD
- الكبح: اضغط على Space

Enjoy the game! 🎮
