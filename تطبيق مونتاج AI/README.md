# MontageAI 🎬

**محرر فيديو بالذكاء الاصطناعي — يعمل بالكامل محلياً على جهازك**

تطبيق مونتاج فيديو احترافي يستخدم الذكاء الاصطناعي لتحرير الفيديو تلقائياً. كل المعالجة تتم **محلياً** على جهازك — بدون اشتراكات، بدون حسابات، بدون رفع ملفات لسيرفرات خارجية.

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| **مونتاج كامل تلقائي** | بضغطة واحدة يحلل الفيديو ويقص ويرتب ويضيف ترجمة وموسيقى |
| **إزالة الصمت** | كشف وحذف التوقفات والصمت من التايملاين |
| **ترجمة تلقائية** | تفريغ الصوت وكتابة النصوص ومزامنتها مع الفيديو |
| **تحسين الصوت** | إزالة الضوضاء وتقوية الترددات وتحسين الوضوح |
| **تحسين الألوان** | تصحيح الإضاءة والتباين والتشبع بفلاتر احترافية |
| **تثبيت الفيديو** | إزالة الاهتزاز وتحسين ثبات المشهد |
| **موسيقى ذكية** | مكتبة مقاطع موسيقية تختار الأنسب حسب المزاج |
| **B-Roll تلقائي** | إضافة مشاهد تكميلية ذكية |
| **Shorts / Reels** | تحويل الفيديو إلى مقاطع عمودية |
| **ترجمة ودبلجة** | ترجمة النصوص لأي لغة ودبلجة بصوت مستنسخ |
| **Thumbnail ذكي** | توليد غلاف فيديو جذاب بالذكاء الاصطناعي |
| **تأثيرات بصرية** | فلاتر لونية وتأثيرات حركية وانتقالات احترافية |
| **تايملاين متعدد المسارات** | فيديو، نص، صوت — سحب وإفلات وتعديل |

---

## 🖥️ التقنيات

- **Next.js 14** — إطار العمل (App Router + TypeScript)
- **TailwindCSS** — التصميم والتنسيق
- **FFmpeg** — معالجة الفيديو والصوت محلياً
- **Web Audio API** — توليد الموسيقى والنغمات محلياً
- **LocalStorage** — حفظ المشاريع على الجهاز

---

## 🚀 التشغيل

```bash
# تثبيت الحزم
pnpm install

# تشغيل خادم التطوير
pnpm dev
```

ثم افتح [http://localhost:3000](http://localhost:3000)

> **ملاحظة:** تحتاج إلى تثبيت برنامج **FFmpeg** على جهازك للمعالجة الكاملة للفيديو.

## 📁 هيكل المشروع

```
montage-ai/
├── app/
│   ├── api/
│   │   ├── ai/           # API endpoints للذكاء الاصطناعي
│   │   ├── export/       # تصدير الفيديو
│   │   ├── ffmpeg/       # معالجة ب FFmpeg
│   │   └── thumbnail/    # توليد الصور المصغرة
│   ├── globals.css       # الأنماط العامة
│   ├── layout.tsx        # التخطيط الرئيسي
│   └── page.tsx          # الصفحة الرئيسية
├── components/
│   └── editor/
│       └── Editor.tsx    # محرر الفيديو الرئيسي
├── lib/
│   ├── types.ts          # أنواع TypeScript
│   ├── project.ts        # إدارة المشاريع (حفظ محلي)
│   └── mockAi.ts         # محاكاة الذكاء الاصطناعي
└── .montage_ai/          # ملفات الفيديو المرفوعة (محلياً)
```

## 🎯 الاستخدام

1. ارفع فيديو (سحب وإفلات أو زر رفع)
2. اختر "مونتاج كامل تلقائي" أو استخدم الأدوات المنفردة
3. حرر وعدّل على التايملاين
4. صدر الفيديو بالجودة والصيغة التي تريدها
